# Fit-and-finish review — cycle 5 (2026-07-03)

Pipeline: vite demo server → Playwright headless Chromium (1440x900, light+dark, scrolling segments of `.preview-wrap`) → Gemini designer critique → Claude agent pixel-crop verification. Findings marked unconfirmed did not survive visual verification but were kept as possible nitpicks.
**Totals: 2 high / 21 medium / 24 low confirmed across 12 templates.**


## ⚠ Blank-render mystery SOLVED (retractions inside)

Five templates captured blank in light mode this cycle. Recapture after fixing the capture script proved these were harness artifacts: `networkidle` fires before the lazy template chunk mounts, so on a cold vite transform the script screenshotted the empty Suspense fallback. The light pass runs first (cold), dark second (warm) — hence the light bias. This retroactively explains the cycle-4 "hydration race" too: it was never a demo-app bug. shoot.mjs now waits for preview content before shooting. The five per-template blank findings are retracted below; their light-mode review coverage is partial this cycle.
